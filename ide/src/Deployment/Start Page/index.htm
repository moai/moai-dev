    Roket3D IDE.NET Released

The first edition of Roket3D IDE.NET has been released. This contains significant improvements over the existing IDE, including proper syntax highlighting, better build tool organisation, powerful builtin editors for models, images, audio, templates, areas and 3D worlds as well as an entirely new framework auto-generated by the IDE and new class system for Lua.

There’s stacks more improvements than just that though, download your copy today to get the most out of the Roket3D game engine.
