<h2>Welcome to Moai! &nbsp;</h2>
Thanks for installing the SDK. &nbsp;Here's the scoop.
<h4>Sample code:</h4>
Go to the samples folder, choose a sample, then execute run.bat (or run.sh) to see it perform. &nbsp;One of our priorities is to build richer samples for Moai developers, so if you'd like to see a particular code sample, let us know by posting in the <a href="http://getmoai.com/forums/">Moai Forums</a>.<br>
<h4>FAQ:</h4>
Answers to frequently asked questions are available in the <a href="http://getmoai.com/marketing/moai-faqs.html">Moai FAQ</a>.<br>
<h4>Wiki:</h4>
You can find detailed documentation, instructions and other helpful information in the <a href="http://getmoai.com/wiki/">Moai Wiki</a>.<br>
<h4>Moai Cloud:</h4>
You already have a Moai Cloud account which allows you to upload Lua scripts to be hosted and run as web services via the <a href="http://dashboard.moaicloud.com/">Moai Cloud Dashboard</a>. &nbsp;If you have support questions,
post them to the <a
href="http://getmoai.com/forums/moai-cloud-developer-support.html">Moai
Cloud Developer Support Forum</a> or check out the <a
href="http://getmoai.com/wiki/">Moai Wiki</a>.<br>
